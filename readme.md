# FrontendTemplate

⚡ Vue3 + TypeScript + Vite Frontend Template ⚡


## Install

```shell
git clone git@github.com:makeryangcom/FrontendTemplate.git
```

## Development

```shell
yarn
```

```shell
yarn dev
```

## Build

```shell
yarn build
```
